# 个人发展与职业规划

> 基于战略方法论的系统化成长指南 —— 从"看清"到"做成"的实战框架

## 项目简介

本项目旨在将宏大的"战略思维"降维，帮助普通人用**"联系"**（横向的社会关系）和**"运动"**（纵向的人生阶段）的观点，解决具体的个人成长问题。

### 核心理念

- **理论结合实际**：从一般到特殊，联系与运动
- **破除孤立主义**：建立个人发展的"联系观"
- **破除静止思维**：建立人生规划的"运动观"

## 主要内容

### 1. 战略方法论五大核心模块

| 模块 | 核心内容 |
|------|----------|
| **战略破局** | 突破与破圈，复利效应，企业生命周期管理 |
| **蓝海战略** | 价值创新，四步动作框架，战略布局坐标图 |
| **竞争战略** | 波特五力模型，三大通用战略（成本领先/差异化/集中化） |
| **战略解码** | RIDER模型，三场会议，硬仗打法 |
| **人才战略** | 3S支撑+3B策略+721法则 |

### 2. 两大根本问题

普通人培养战略思维只需要问自己两个问题：

1. **这个决定对周围的人和事有什么影响？** （联系观）
2. **这个决定对未来的我有什么影响？** （运动观）

### 3. 服务产品体系

**方法论：调查分析 → 指导方案 → 执行优化（DPO模型）**

1. **战略诊断**：环境分析 / 能力评估 / 机会威胁识别
2. **战略规划**：目标设定 / 路径设计 / 阶段里程碑
3. **执行落地**：行动计划 / 跟踪辅导 / 反馈调整

## 快速开始

### 查看在线指南

直接在浏览器中打开 `index.html` 文件，即可查看完整的个人发展与职业规划战略指南。

```bash

git clone https://github.com/wanglin1111111/career-development.git

# 进入目录
cd career-development

# 打开网页（Windows）
start index.html

# 打开网页（Mac）
open index.html

# 打开网页（Linux）
xdg-open index.html
```
